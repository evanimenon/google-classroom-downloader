Privacy Policy

Classroom Downloader (“we”, “our”, “this app”) provides a tool that allows users to download their own Google Classroom coursework and files directly to their personal device.

What data we access

We access only the minimum Google account data required to perform downloads:

  • Google profile information (name, email)
  • Google Classroom courses and assignments
  • Student submissions and attached Drive files

How data is used

Data is used strictly to allow the authenticated user to view and download their own coursework files.

Data storage

No personal data is permanently stored. Files are processed in memory and immediately delivered to the user. Temporary data is automatically deleted within minutes.

Data sharing

We do not share, sell, or distribute any user data to any third party.

Security

All communication is encrypted using HTTPS. Access tokens are stored only in memory and never persisted.

Data deletion

All data is deleted automatically after each session. No manual deletion is required.
