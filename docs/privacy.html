Privacy Policy

Google Classroom Downloader is a read-only educational utility that allows users to download their own Google Classroom coursework and attachments.

Data Accessed

  • Basic Google profile information (name, email)
  • Google Classroom courses and assignments
  • Student submissions and attached Google Drive files

How Data Is Used

Data is accessed only to allow the authenticated user to view and download their own coursework files.

Data Storage

No personal data is permanently stored. Files are processed temporarily and immediately delivered to the user. Temporary session data is automatically deleted.

Data Sharing

We do not sell, share, or distribute user data to any third party.

Security

All communication is encrypted using HTTPS.

This application is an independent third-party service and is not affiliated with or endorsed by Google.
